# Support

Use the following channel:

| Topic | Channel |
| --- | --- |
| Reproducible incorrect behavior | [Bug report](https://github.com/makcimbx/opencode-better-hashline/issues/new?template=bug.yml) |
| Feature or protocol proposal | [Feature request](https://github.com/makcimbx/opencode-better-hashline/issues/new?template=feature.yml) |
| Configuration or usage question | [GitHub Discussions](https://github.com/makcimbx/opencode-better-hashline/discussions) |
| Security vulnerability | [Private vulnerability report](https://github.com/makcimbx/opencode-better-hashline/security/advisories/new) |

Support covers the compatibility matrix in the README. Include sanitized `opencode --version`, OS,
installation method, plugin options, and a minimal reproduction.
